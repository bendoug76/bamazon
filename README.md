# Bamazon

A storefront app using MySQL, and Node.js 

* To use as a customer:
    * npm install 
        * mysql
        * prompt
    * Run node bamazonCustomer.js
    * Follow command-line prompts.
    *

* To use as a manager:
    * npm install:
        * mysql
        * prompt
        * inquirer
    * Run node bamazonManager.js
    * Follow command-line prompts.
